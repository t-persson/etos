# ETOS API

The launcher of ETOS.

Github: [ETOS API](https://github.com/eiffel-community/etos-api)

The ETOS API is a python microservice running on [FastAPI](https://fastapi.triangolo.com) that provides the following functionalities:

- Generate and execute and ETOS test suite. (Sending [TERCC](../concepts/tercc.md) to ETOS)
